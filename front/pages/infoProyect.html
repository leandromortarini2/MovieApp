informacion del proyecto

Index / peliculas

En esta pagina las peliculas son solicitadas de la base de datos, las etiquetas de las cards fueron creadas con DOM y las cards con css puro. Tambien se utilizo bootstrap para los botones.

anime / series

Los datos de cada una de las cards fue buscado individualmente, y se le dieron los enlaces reales de donde de pueden encontrar y/o ver cada uno de los titulos.
las tarjetas fueron creadas con boostrap.

Creador de Pelis

Este form envia los datos necesarios para la creacion de nuevas cards en el inicio de la pagina. La informacion de cada input es convertida en un objeto y enviada al back end para formar la peli y asi guardarce en la base de datos.

responsive

El responsive de la pagina fue creado con css puro de media query, pagina por pagina, seccion por seccion, facilitando la tarea por la utilizacion de la tecnica de box model.
